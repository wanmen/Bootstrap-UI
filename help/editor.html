我们采用 PageDown 来作为 Markdown 解析器。Markdown 是一种轻量级标记语言，你可以参阅这份说明文档。我们使用 MathJax 来支持 LaTeX 数学公式，比如

\( J_\alpha(x) = \sum\limits_{m=0}^\infty \frac{(-1)^m}{m! \, \Gamma(m + \alpha + 1)}{\left({\frac{x}{2}}\right)}^{2 m + \alpha} \)

你可以在下面学习 Markdown 语法。注意，数学公式的动态预览目前还不支持。在 Firefox 浏览器中，你可以通过刷新页面来预览 LaTeX 输出。
